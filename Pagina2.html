Inventario

Nuestro sistema de base de datos se basa en Microsoft SQL Server el cual es un sistema de bases de datos racional el cual permite almacenar, modificar y consultar datos por medio del lenguaje SQL

Características de nuestra base de datos:

  • Manejamos una base de datos organizada la cual facilita la gestion y recuperacion ediciente de datos,
  • Nuestra integridad de datos consta de una gran seguridad, datos precisos y consistentes
  • Además constamos de la capacidad de relacionar datos de manera logica.

Para obtener más información, puedes consultar la sitio web de MySQL.
